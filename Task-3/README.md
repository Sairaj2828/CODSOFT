# Sales Prediction Using Python 📊

## 📌 Project Overview
Sales prediction is the process of forecasting future sales based on historical data and advertising spending.

In this project, machine learning is used to analyze advertising budgets across multiple platforms to predict product sales.

---

## 📊 Dataset

The dataset contains advertising spending across different platforms.

### Features
- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget

### Target Variable
- Sales

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

---

## 🤖 Machine Learning Model

The model used for prediction:

- Linear Regression

---

## ⚙️ Steps Performed
1. Import required libraries
2. Load dataset
3. Select features and target variable
4. Split dataset into training and testing sets
5. Train Linear Regression model
6. Predict sales values
7. Evaluate model performance

---

## 📈 Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

---

## 🎯 Result
The model predicts product sales based on advertising spending and helps understand how marketing budgets affect sales performance.
