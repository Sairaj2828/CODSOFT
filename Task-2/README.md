# Iris Flower Classification 🌸

## 📌 Project Overview
The Iris Flower Classification project predicts the species of an iris flower using machine learning based on flower measurements.

This is one of the most popular beginner datasets used for classification problems.

---

## 📊 Dataset

The dataset includes measurements of iris flowers.

### Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### Target Variable
- Species

### Species Classes
- Iris Setosa
- Iris Versicolor
- Iris Virginica

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

---

## 🤖 Machine Learning Model

The model used in this project:

- Random Forest Classifier

---

## ⚙️ Steps Performed
1. Import libraries
2. Load the dataset
3. Select features and target variable
4. Encode categorical labels
5. Split dataset into training and testing sets
6. Train the model
7. Make predictions
8. Evaluate the model

---

## 📈 Evaluation Metrics
- Accuracy Score
- Classification Report

---

## 🎯 Result
The model accurately classifies iris flowers into their correct species based on petal and sepal measurements.
